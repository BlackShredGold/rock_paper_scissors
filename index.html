Rock, Paper Scissors

Right now this game is only playable in the developer console. Here are some notes about how it plays:

  1. The game plays five total rounds against the computer (not best of five).
  2. If you tie with the computer, you get to do that round over.
  3. The program accepts "rock," "paper," or "scissors" as input. Input is not case sensitive.
  4. If you input any text other than the above, you will get an error and be prompted to try the round again.

Have fun!
